# Auto-Map (live, Trizbort-style grid) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** As the author plays interactively, draw a live Trizbort-style map of discovered rooms and their connections in the (currently stubbed) Map tab, with zoom/pan/fit and a per-room hover popover showing exits + objects.

**Architecture:** A `MutationObserver` in the Play iframe posts the room name (status line) + this turn's new buffer text each turn. `PlayPanel` feeds a `useMap` store, which builds a pure graph and a deterministic grid `layout`. `MapPanel` renders it as zoomable SVG. The map is session state, scoped per game and reset on compile — exactly like the play transcript.

**Tech Stack:** Nuxt 4 (`ssr:false`, client-only), Vue 3 `<script setup>`, Nuxt UI 4, TypeScript strict, Vitest (node for pure logic; happy-dom for composables, per the existing Tier-B setup).

**Spec:** `docs/superpowers/specs/2026-06-30-auto-map-design.md`

## Global Constraints

- **No `Co-Authored-By` / AI-attribution trailer** on any commit.
- **Pure logic is TDD'd in the node env** (`app/composables/map-graph.ts` → `map-graph.test.ts`). Composable tests use the repo's happy-dom setup (same as `useIde`/`usePlayTranscript` tests).
- **Per-game + clean slate:** the map is NOT persisted; it resets on `activeStoryKey` change and on compile, like `usePlayTranscript`. Read the story key via `useState('frotz:story-key')` **directly** (do NOT call `useIde()` from `useMap` — that re-enters `useIde` and risks a cycle; the other scoped composables read the key directly).
- **postMessage trust boundary unchanged:** `PlayPanel.onMessage` already gates on `e.origin === location.origin`, `e.source === playFrame.contentWindow`, and `data.source === 'frotzsmith-play'`. Room data is inert text — rendered via SVG `<text>`/escaped Vue interpolation, never HTML.
- **Capture suppressed during scripted runs:** the iframe capture must no-op while `window.__frotzScriptRunning` is set (consistent with the transcript).
- **Accessibility:** the map is keyboard-operable (focusable rooms, button zoom controls) and axe-clean (repo baseline).
- Follow existing patterns: composables in `app/composables/`, pure helpers as a sibling `*.ts` with a `*.test.ts`, components in `app/components/ide/`.

```ts
// app/composables/map-graph.ts
export type Dir = 'n' | 's' | 'e' | 'w' | 'ne' | 'nw' | 'se' | 'sw' | 'u' | 'd' | 'in' | 'out'
export interface Edge { from: string; to: string; dir: Dir }
export interface MapGraph { rooms: string[]; edges: Edge[]; start: string | null }

export function emptyGraph(): MapGraph {
  return { rooms: [], edges: [], start: null }
}

const DIRS: Record<string, Dir> = {
  n: 'n', north: 'n', s: 's', south: 's', e: 'e', east: 'e', w: 'w', west: 'w',
  ne: 'ne', northeast: 'ne', nw: 'nw', northwest: 'nw',
  se: 'se', southeast: 'se', sw: 'sw', southwest: 'sw',
  u: 'u', up: 'u', d: 'd', down: 'd',
  in: 'in', inside: 'in', enter: 'in', out: 'out', outside: 'out', exit: 'out',
}

/** Parse a command to a movement direction, else null. n/north/'go north'/ne/up/in/out. */
export function parseDirection(command: string): Dir | null {
  const c = command.trim().toLowerCase().replace(/^(go|walk|run|head)\s+/, '')
  return DIRS[c] ?? null
}

/** Room name from a status-line string: the left text before the score/moves gap. */
export function parseRoomName(statusLine: string): string {
  const firstLine = statusLine.split('\n')[0] ?? ''
  return firstLine.split(/\s{2,}/)[0]!.trim()
}

/** Record a turn: always ensures `newRoom` exists; adds an edge prev→new when
 *  `dir` is set and the room actually changed (failed moves add nothing). */
export function addStep(g: MapGraph, prevRoom: string | null, dir: Dir | null, newRoom: string): MapGraph {
  if (!newRoom) return g
  const rooms = g.rooms.includes(newRoom) ? g.rooms : [...g.rooms, newRoom]
  const start = g.start ?? newRoom
  let edges = g.edges
  if (dir && prevRoom && prevRoom !== newRoom) {
    const exists = edges.some(e => e.from === prevRoom && e.to === newRoom && e.dir === dir)
    if (!exists) edges = [...edges, { from: prevRoom, to: newRoom, dir }]
  }
  return { rooms, edges, start }
}
```

```ts
export interface PlacedRoom { name: string; col: number; row: number }
export interface Connector { from: string; to: string; dir: Dir; grid: boolean }
export interface MapLayout {
  rooms: PlacedRoom[]
  connectors: Connector[]
  bounds: { minCol: number; maxCol: number; minRow: number; maxRow: number }
}

const DELTA: Record<Dir, [number, number]> = {
  n: [0, -1], s: [0, 1], e: [1, 0], w: [-1, 0],
  ne: [1, -1], nw: [-1, -1], se: [1, 1], sw: [-1, 1],
  u: [0, 0], d: [0, 0], in: [0, 0], out: [0, 0],
}

/** Trizbort grid placement (deterministic; insertion order; bounded spiral fallback). */
export function layout(g: MapGraph): MapLayout {
  const pos = new Map<string, [number, number]>()
  const occupied = new Set<string>()
  const key = (c: number, r: number) => `${c},${r}`
  const place = (room: string, c: number, r: number) => { pos.set(room, [c, r]); occupied.add(key(c, r)) }
  const nearestFree = (c: number, r: number): [number, number] => {
    for (let ring = 0; ring < 64; ring++)
      for (let dc = -ring; dc <= ring; dc++)
        for (let dr = -ring; dr <= ring; dr++) {
          if (Math.max(Math.abs(dc), Math.abs(dr)) !== ring) continue
          if (!occupied.has(key(c + dc, r + dr))) return [c + dc, r + dr]
        }
    return [c, r]
  }

  if (g.start) place(g.start, 0, 0)

  // Repeated passes place each edge's target next to its already-placed source.
  let changed = true
  while (changed) {
    changed = false
    for (const e of g.edges) {
      const from = pos.get(e.from)
      if (!from || pos.has(e.to)) continue
      const [dc, dr] = DELTA[e.dir]
      if (dc === 0 && dr === 0) {
        const [c, r] = nearestFree(from[0], from[1]); place(e.to, c, r)
      } else if (!occupied.has(key(from[0] + dc, from[1] + dr))) {
        place(e.to, from[0] + dc, from[1] + dr)
      } else {
        const [c, r] = nearestFree(from[0] + dc, from[1] + dr); place(e.to, c, r)
      }
      changed = true
    }
  }
  // Orphans unreachable from start: drop them at free cells, insertion order.
  for (const room of g.rooms) if (!pos.has(room)) { const [c, r] = nearestFree(0, 0); place(room, c, r) }

  const connectors: Connector[] = g.edges.map(e => {
    const a = pos.get(e.from)!, b = pos.get(e.to)!
    const [dc, dr] = DELTA[e.dir]
    const grid = (dc !== 0 || dr !== 0) && b[0] === a[0] + dc && b[1] === a[1] + dr
    return { from: e.from, to: e.to, dir: e.dir, grid }
  })

  const placed = [...pos.values()]
  const cols = placed.map(p => p[0]); const rows = placed.map(p => p[1])
  const bounds = placed.length
    ? { minCol: Math.min(...cols), maxCol: Math.max(...cols), minRow: Math.min(...rows), maxRow: Math.max(...rows) }
    : { minCol: 0, maxCol: 0, minRow: 0, maxRow: 0 }

  const rooms: PlacedRoom[] = g.rooms.map(name => ({ name, col: pos.get(name)![0], row: pos.get(name)![1] }))
  return { rooms, connectors, bounds }
}
```

```ts
const DIR_ORDER: Dir[] = ['n', 'ne', 'e', 'se', 's', 'sw', 'w', 'nw', 'u', 'd', 'in', 'out']

/** Directions of edges leading out of `room`, in canonical compass order. */
export function exitsOf(g: MapGraph, room: string): Dir[] {
  const dirs = new Set<Dir>()
  for (const e of g.edges) if (e.from === room) dirs.add(e.dir)
  return DIR_ORDER.filter(d => dirs.has(d))
}

function splitObjects(s: string): string[] {
  return s
    .split(/\s*,\s*|\s+and\s+/i)
    .map(x => x.replace(/^\s*(?:a|an|the|some)\s+/i, '').trim())
    .filter(Boolean)
}

/** Best-effort object names from a room's captured text (Std/Puny "X here" listing). */
export function parseObjects(roomText: string): string[] {
  const out: string[] = []
  for (const line of roomText.split('\n')) {
    const see = /you can (?:also )?see (.+?) here\b/i.exec(line)
    if (see) out.push(...splitObjects(see[1]!))
    const there = /there (?:is|are) (.+?) here\b/i.exec(line)
    if (there) out.push(...splitObjects(there[1]!))
  }
  return out
}
```

```ts
// app/composables/useMap.ts
import {
  emptyGraph, addStep, layout as computeLayout, parseDirection, exitsOf, parseObjects,
  type Dir, type MapGraph,
} from './map-graph'

function firstLine(t: string): string {
  return t.split('\n').map(l => l.trim()).find(Boolean) ?? ''
}

// Register the cross-game reset watcher exactly once (see the note in useMap).
let mapWatchRegistered = false

export function useMap() {
  // Read the active story key directly (do NOT call useIde() — avoid re-entrancy).
  const activeStoryKey = useState<string>('frotz:story-key', () => '')
  const graph = useState<MapGraph>('frotz:map-graph', emptyGraph)
  const currentRoom = useState<string | null>('frotz:map-current', () => null)
  const roomText = useState<Record<string, string>>('frotz:map-roomtext', () => ({}))
  const lastDir = useState<Dir | null>('frotz:map-lastdir', () => null)
  const prevRoom = useState<string | null>('frotz:map-prevroom', () => null)

  function reset() {
    graph.value = emptyGraph()
    currentRoom.value = null
    roomText.value = {}
    lastDir.value = null
    prevRoom.value = null
  }

  function recordCommand(cmd: string) {
    lastDir.value = parseDirection(cmd)
    prevRoom.value = currentRoom.value
  }

  function recordRoom(name: string, text: string) {
    if (!name) return
    if (text) roomText.value = { ...roomText.value, [name]: text }
    graph.value = addStep(graph.value, prevRoom.value, lastDir.value, name)
    currentRoom.value = name
    lastDir.value = null // a later redraw without a fresh command must not draw an edge
  }

  function details(room: string) {
    const t = roomText.value[room] ?? ''
    return { exits: exitsOf(graph.value, room), objects: parseObjects(t), description: firstLine(t) }
  }

  const layout = computed(() => computeLayout(graph.value))

  // Reset when the active story changes (matches usePlayTranscript scoping).
  // Register ONCE — useMap() is also called from runCompile (non-setup) and from
  // several components; a per-call watch would leak/duplicate. Mirror whatever
  // approach usePlayTranscript uses; a module-level flag is the simplest. Shared
  // useState means the first closure's reset() still clears the shared state.
  if (import.meta.client && !mapWatchRegistered) {
    mapWatchRegistered = true
    watch(activeStoryKey, reset)
  }

  return { graph, currentRoom, roomText, layout, recordCommand, recordRoom, details, reset }
}
```

```js
// --- Auto-map capture: post the room (status line) + this turn's new buffer text ---
var __frotzPrevBufferLen = 0
var __frotzLastRoom = ''
function __frotzReadRoom() {
  if (window.__frotzScriptRunning) return
  var grid = document.querySelector('.GridWindow')
  if (!grid) return
  var line0 = (grid.textContent || '').split('\n')[0] || ''
  var name = line0.split(/\s{2,}/)[0].trim()
  if (!name) return
  var buffer = document.querySelector('.BufferWindow')
  var full = buffer ? (buffer.textContent || '') : ''
  // Buffer accumulates; the suffix added since last read is this turn's output.
  if (full.length < __frotzPrevBufferLen) __frotzPrevBufferLen = 0 // a screen clear
  var text = full.slice(__frotzPrevBufferLen)
  // Only post when the room or the buffer changed (coalesce observer bursts).
  if (name === __frotzLastRoom && text.length === 0) return
  __frotzPrevBufferLen = full.length
  __frotzLastRoom = name
  parent.postMessage({ source: 'frotzsmith-play', type: 'room', name: name, text: text }, location.origin)
}
var __frotzRaf = 0
function __frotzScheduleRead() {
  if (__frotzRaf) return
  __frotzRaf = requestAnimationFrame(function () { __frotzRaf = 0; __frotzReadRoom() })
}
function __frotzStartMapObserver() {
  var root = document.getElementById('windowport') || document.body
  new MutationObserver(__frotzScheduleRead).observe(root, { childList: true, subtree: true, characterData: true })
  __frotzReadRoom() // initial room before any command
}
// Start once GlkOte has built the windows. If the command hook already waits for
// GlkOte readiness, start there; otherwise poll briefly:
;(function waitForWindows() {
  if (document.querySelector('.GridWindow')) __frotzStartMapObserver()
  else setTimeout(waitForWindows, 120)
})()
```

- [ ] **Step 1: Base SVG render (`MapPanel.vue`)**

`viewBox` here is a static fit to `bounds` (interactivity arrives in Task 8). Constants: `CELL = 120`, `ROOM_W = 96`, `ROOM_H = 48`, `PAD = 60`. Room centre = `(col*CELL, row*CELL)`.

```vue
<script setup lang="ts">
import type { Dir } from '~/composables/map-graph'
const { layout, currentRoom } = useMap()

const CELL = 120, ROOM_W = 96, ROOM_H = 48, PAD = 60
const hasRooms = computed(() => layout.value.rooms.length > 0)
const noMap = computed(() => false) // Task 8/later may surface a "no room name" note

const viewBox = computed(() => {
  const b = layout.value.bounds
  const x = b.minCol * CELL - ROOM_W / 2 - PAD
  const y = b.minRow * CELL - ROOM_H / 2 - PAD
  const w = (b.maxCol - b.minCol) * CELL + ROOM_W + PAD * 2
  const h = (b.maxRow - b.minRow) * CELL + ROOM_H + PAD * 2
  return `${x} ${y} ${w} ${h}`
})
const pos = (name: string) => {
  const r = layout.value.rooms.find(x => x.name === name)!
  return { x: r.col * CELL, y: r.row * CELL }
}
const dirLabel: Record<Dir, string> = { n:'N', s:'S', e:'E', w:'W', ne:'NE', nw:'NW', se:'SE', sw:'SW', u:'↑', d:'↓', in:'in', out:'out' }
</script>

<template>
  <div class="bg-default relative h-full w-full overflow-hidden">
    <div v-if="!hasRooms" class="frotz-grid flex h-full flex-col items-center justify-center gap-3 p-6 text-center">
      <UIcon name="i-lucide-map" class="size-10 text-primary" />
      <p class="text-lg font-semibold">Play to map the world</p>
      <p class="text-muted max-w-sm text-sm">Move around in the Play tab and rooms appear here.</p>
    </div>

    <svg v-else class="h-full w-full" :viewBox="viewBox" preserveAspectRatio="xMidYMid meet"
         role="img" :aria-label="`Map: ${layout.rooms.length} rooms`">
      <!-- connectors -->
      <g stroke="currentColor" class="text-muted">
        <line v-for="(c, i) in layout.connectors" :key="i"
              :x1="pos(c.from).x" :y1="pos(c.from).y" :x2="pos(c.to).x" :y2="pos(c.to).y"
              :stroke-dasharray="c.grid ? '0' : '6 5'" stroke-width="2" />
      </g>
      <!-- rooms -->
      <g v-for="r in layout.rooms" :key="r.name" :transform="`translate(${r.col * CELL},${r.row * CELL})`">
        <rect :x="-ROOM_W/2" :y="-ROOM_H/2" :width="ROOM_W" :height="ROOM_H" rx="8"
              class="fill-elevated"
              :class="r.name === currentRoom ? 'stroke-amber-500' : 'stroke-default'"
              :stroke-width="r.name === currentRoom ? 3 : 1.5" />
        <text text-anchor="middle" dominant-baseline="middle" class="fill-default text-sm">{{ r.name }}</text>
      </g>
    </svg>
  </div>
</template>
```
(If a room name overflows `ROOM_W`, truncate visually with a `<title>` for the full name — acceptable to defer to Task 9's popover.)

### Task 8: Zoom / pan / fit controls

- [ ] **Step 1: Make the `viewBox` interactive**

Replace the static `viewBox` computed with reactive state `view = ref({ x, y, w, h })`, an `auto` flag (true until the user interacts), and:
- `fitView()` — set `view` to the bounds-based box (the old computed's value). Called on first rooms and by the **Fit** button. While `auto` is true, re-fit whenever `layout` grows (watch `layout`), so the map stays fully visible until the user zooms/pans.
- `zoomBy(factor, cx?, cy?)` — scale `w`/`h` by `factor` about a centre (default the viewport centre), clamp to sensible min/max (e.g. `w` in `[CELL, CELL*60]`); sets `auto=false`. **Zoom in** = `zoomBy(0.8)`, **Zoom out** = `zoomBy(1.25)`.
- wheel handler on the SVG → `zoomBy(e.deltaY<0?0.9:1.1, mouseSvgX, mouseSvgY)` (convert client→SVG coords via `getBoundingClientRect` + current `view`); `preventDefault`.
- drag-pan: `pointerdown`/`move`/`up` translate `view.x/y` by the SVG-scaled delta; `auto=false`.
- `zoomPct = computed(() => Math.round(baseW / view.value.w * 100))` where `baseW` is the fit width — a readout.
- keep current room on-screen: `watch(currentRoom, …)` — if the current room centre is outside `view`, pan `view` to bring it in (only when `!auto`, so an explicit zoom isn't yanked away unexpectedly; when `auto`, the re-fit already covers it).
